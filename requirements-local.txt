#
#   # Step 1: torch CPU 版を先にインストール（必須・順番厳守）
#   pip install torch==2.4.1 torchaudio==2.4.1 --index-url https://download.pytorch.org/whl/cpu
#
#   # Step 2: このファイルをインストール（torch が入った後なら PyPI 版を引かない）
#   pip install -r requirements.txt
#   pip install -r requirements-local.txt
#
# Docker で使う場合: Dockerfile.diarize が上記手順を正しい順で自動実行する
# ===================================
